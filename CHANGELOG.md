## 0.0.2

### 新增

- 截图支持矩形标注，可选择颜色和线宽，并支持撤销；标注会随图片一起保存或复制。
- 设置页新增分类页签，并在关于页显示当前版本、支持手动检查更新。

### 优化

- 重做截图工具栏，保存、复制、取消和二维码识别等操作更加直观。
- 优化应用内更新提示和安装流程。

### 修复

- 修复 macOS 截图覆盖层可能出现白色背景或白块的问题。
- 修复应用启动与快捷键同时触发时，截图窗口可能初始化失败的问题。
- 修复截图选区靠近屏幕边缘时可能出现的裁剪越界问题。

## 0.0.1

sbox 是一个多功能桌面工具箱，聚合了日常开发、内容处理和账号辅助中常用的小工具，适合快速完成格式转换、数据校验、图片处理、截图和接入信息导出等任务。

### 主要功能

- 格式处理：支持 Base64、URL、Hex、HTML、Unicode、JSON 等常见编码转换。
- 数据查看：支持 JSON 格式化、压缩、排序、树状查看，以及 JSON / YAML / TOML 互转。
- 开发辅助：支持 JWT 解码、时间戳转换、时区换算、哈希计算和文件校验。
- 生成工具：支持生成密码、UUID、随机字符串、二维码和 Android keystore。
- 图片媒体：支持二维码识别、图片格式转换、批量处理、裁剪、旋转和尺寸调整。
- 桌面能力：支持框选截图、保存截图、复制到剪贴板和全局快捷键。
- 账号辅助：支持小米小爱和 Google Drive 授权信息导出。

### 定位

sbox 希望把零散的小工具集中到一个干净、易用的桌面应用里，减少在多个网页、脚本和临时工具之间切换的成本。
