# Codec

A grab-bag of everyday encode / decode, hashing and generator utilities — all local, nothing leaves your machine.

## Conversions

| Type | Directions |
| --- | --- |
| **Base64** | encode / decode |
| **URL** | encode / decode |
| **Hex** | encode / decode |
| **HTML entities** | escape / unescape |
| **Unicode** | escape (`\uXXXX`) / unescape |
| **JSON** | format / minify |

## Hashes

Compute a digest of the input text:

- **MD5**
- **SHA-1**
- **SHA-256**
- **SHA-512**

## Generators & helpers

- **UUID** — generate random identifiers.
- **Timestamp** — convert between Unix timestamps and human-readable dates.
- **Byte size** — format raw byte counts as human-readable sizes.

## Usage

1. Open sbox and choose **Codec**.
2. Pick the operation, paste or type your input.
3. Copy the result.

Useful for quickly decoding a token, hashing a string, or pretty-printing a JSON payload without leaving the toolbox.
