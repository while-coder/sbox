# 编解码工具

一组日常用的编解码、哈希与生成器工具 —— 全部本地运行，数据不出本机。

## 转换

| 类型 | 方向 |
| --- | --- |
| **Base64** | 编码 / 解码 |
| **URL** | 编码 / 解码 |
| **Hex** | 编码 / 解码 |
| **HTML 实体** | 转义 / 反转义 |
| **Unicode** | 转义（`\uXXXX`）/ 反转义 |
| **JSON** | 格式化 / 压缩 |

## 哈希

对输入文本计算摘要：

- **MD5**
- **SHA-1**
- **SHA-256**
- **SHA-512**

## 生成器与辅助

- **UUID** —— 生成随机标识符。
- **时间戳** —— 在 Unix 时间戳与可读日期之间转换。
- **字节大小** —— 把原始字节数格式化为可读大小。

## 使用方式

1. 打开 sbox，选择 **编解码工具**。
2. 选择操作，粘贴或输入内容。
3. 复制结果。

适合快速解码一个 token、对字符串求哈希，或在不离开工具箱的情况下美化一段 JSON。
