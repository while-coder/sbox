# 小爱登录

登录小米账号，导出 sbot `channel.xiaoai` 与小爱音箱通信所需的凭据。

## 你会拿到

| 字段 | 来源 |
| --- | --- |
| `userId` | 小米账号 ID |
| `passToken` | 长期登录令牌 |
| `deviceId` | 登录设备标识 |

以及账号下**每一台音箱**的：

| 字段 | 说明 |
| --- | --- |
| `name` | 音箱型号名称 |
| `deviceID` | 音箱设备 ID |
| `miotDID` | MIoT 设备 ID |

## 操作步骤

1. 打开 sbox，选择 **小爱登录**。
2. 点击 **打开小米登录**，会弹出小米官方登录页。
   - 支持账号密码 / 短信 / 扫码 / 图形验证，按小米页面引导操作即可。
   - 如果点击后自动登上了上次的账号，先点 **清除登录缓存** 再试。
3. 登录成功后，sbox 会自动拉取音箱列表。
4. 每一项都有独立的 **复制** 按钮，按需复制。

## 说明

- 如果登录成功但拉取音箱列表失败，账号凭据仍然可用，重试拉取即可。
- 若账号下没有小爱音箱，会提示「该账号下未发现小爱音箱」。
- 用 **清除登录缓存** / **切换账号** 可以换一个小米账号登录。

## 在 sbot 中使用

把对应字段填进 `channel.xiaoai` 配置（或粘贴到 sbot admin）。至少需要 `userId`、`passToken`、`deviceId`，以及目标音箱的 `deviceID` / `miotDID`。
